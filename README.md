# AdaptiveHome

> Make your home adaptive!!


## How tu run
```
git clone https://github.com/jacopoRodeschini/AdaptiveHome.git
cd AdaptiveHome/src
```

```
npm install 
SESSION_SECRET="secret_key" node bin/www.js
```


## executions

Open Google-Chrome and follow the link: 

```
http://localhost:3000/ 
```
